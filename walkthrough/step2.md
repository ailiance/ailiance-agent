# AST Precision

**ailiance-agent understands language syntax and can read from and manipulate a language's syntax directly.**

Instead of simple text matching, ailiance-agent uses AST-aware tools to target specific symbols (like classes or functions). This ensures that edits are syntactically valid, braces are always balanced, and metadata like JSDoc or decorators are handled automatically.
