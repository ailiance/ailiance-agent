# Hash Anchored MultiFile Edits

**ailiance-agent targets line hashes to perform high accuracy code edits and can batch multiple edits in a single file and multiple files in one shot.**

By using a deterministic line-hash protocol, ailiance-agent eliminates the "friction" of standard AI coding. It locks onto the exact lines regardless of how the file has shifted, allowing for complex global refactors in a single roundtrip.
